FreeCodeCamp API Basejump: Timestamp Microservice

User stories:
  • I can pass a string as a parameter, and it ill check to see whether that string contains a unix timestamp or a natural language date (example: December 25, 2016)
  • If it does, it returns both the unix timestamp and the natural language form of the date.
  • If it does not contain a unix timestamp or natural language date, it returns null
